Услуги

Консультации по компьютерной безопасности

Услуги уровня middle–senior: фокус на управлении рисками, инженерных контролях и понятных результатах (отчёт, roadmap, рекомендации по приоритизации и quick wins).

  • Аудит и диагностика: posture review, gap-analysis, hardening, анализ конфигураций, логирования, прав доступа.
  • AppSec / Secure SDLC: threat modeling, security code review, SAST/DAST, управление секретами, безопасные релизы.
  • Стандарты и практики: OWASP Top 10 / ASVS, NIST CSF/800-53 (по необходимости), базовые подходы ISO 27001.
  • Инфра и сеть: IAM (least privilege), сегментация, TLS/PKI, WAF, базовые меры для cloud/on-prem.
  • IR-готовность: рекомендации по мониторингу, алертингу, журналированию, runbooks, tabletop-сценарии.

Тулчейн (без “магии”, по делу): Burp Suite / OWASP ZAP, Semgrep/SonarQube, GitHub Actions/GitLab CI, Docker, базово Terraform, логирование/метрики (в зависимости от стека).

Разработка сайтов (визитки/лендинги) — middle–senior

Делаю сайты, которые выглядят нормально и ведут себя нормально: адаптив, скорость, SEO-база, аналитика, чистые интеграции и аккуратный деплой.

  • Фронтенд: HTML/CSS, JS/TS, компонентный подход, доступность (a11y), Core Web Vitals.
  • Бэкенд (если нужно): Node.js / Python (FastAPI), REST, формы/почта, простые интеграции.
  • Данные: PostgreSQL (по задаче), базовые кеш-подходы (Redis при необходимости).
  • Деплой: Nginx, Docker, CI/CD, домен/HTTPS, эксплуатационные мелочи (логи, бэкапы, env-секреты).
  • Практика: код-ревью, понятные коммиты, чек-листы, документация “как поддерживать”.

Если нужен максимально простой вариант — делаю статический сайт без тяжёлых зависимостей. Если нужна “чуть сложнее” — подключаю бэкенд/интеграции и нормальный пайплайн.

Обо мне

Опыт в IT 10+ лет. Работаю на стыке инженерии и безопасности: умею разложить задачу по рискам, собрать адекватные меры, объяснить “почему так”, и довести до внедрения. В веб-разработке придерживаюсь практик, которые не стыдно поддерживать: качество, производительность, предсказуемый деплой.

  • Инженерный уровень: архитектурные решения, приоритизация, trade-offs, документация, сопровождение.
  • Security-мышление: модели угроз, контроль поверхности атаки, принципы least privilege и defense-in-depth.
  • Процесс: чек-листы, PR/ревью, CI/CD, базовая автоматизация, прозрачные договорённости по результату.
